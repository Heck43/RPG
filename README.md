# RPG на C++ (задание для контрольной)

Консольная пошаговая RPG-игра. Игрок сражается с врагами, используя оружие, броню и зелья.

## Структура классов

Игра разделена на несколько классов, связанных через наследование и композицию.

### Иерархия персонажей

- **Character** — базовый класс. Хранит имя, HP (текущее и максимальное), базовую атаку и защиту. Используется как родитель для игрока и врага.

- **Player** (наследуется от Character) — класс игрока. Может экипировать оружие и броню, носить зелья в инвентаре. Финальная атака и защита считаются с учётом снаряжения.

- **Enemy** (наследуется от Character) — класс врага. Дополнительно хранит тип врага (например, «Гоблин»). Атакует автоматически.

### Предметы

- **Item** — базовый класс для всех предметов. Хранит название и описание.

- **Weapon** (наследуется от Item) — оружие, добавляет бонус к атаке.

- **Armor** (наследуется от Item) — броня, добавляет бонус к защите.

- **Potion** (наследуется от Item) — зелье лечения, восстанавливает HP игроку.

### Игровые системы

- **BattleSystem** — управляет пошаговым боем. На каждом ходу игрок выбирает действие: атаковать, использовать зелье или осмотреть снаряжение. После хода игрока атакует враг.

- **main.cpp** — точка входа. Создаёт игрока с базовым снаряжением и зельями, создаёт врага (Гоблин-воин), запускает бой через BattleSystem.

## Используемые концепции C++

- Наследование и виртуальные деструкторы
- Динамическое выделение памяти (`new` / `delete`)
- Инкапсуляция (геттеры, приватные поля)
- Полиморфизм через базовые классы
